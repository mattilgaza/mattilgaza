Mat til Gaza

Denne README.md filen inneholder informasjon om utviklingen av Mat til Gaza's hjemmeside.

## Struktur

MATTILGAZA
├── nextjs-mat-til-gaza  # Frontend React app
├── studio-mat-til-gaza  # Sanity Content Studio
└── README.md

##### Frontend React app
    - Brukergrensesnitt for Mat til Gaza's hjemmeside. Her bestemmes hvordan hjemmesiden er satt opp.

##### Sanity Content Studio
    - Plattform for å lage og redigere innhold for Mat til Gaza's hjemmeside. Her bestemmes hvilket innhold som er tilgjengelig på hjemmesiden. Ideen er at innhold enkelt kan redigeres av ikke-tekniske brukere.
